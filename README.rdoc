== PAKUTÀ SOCIAL NETWORK
* PAKUTÀ is a web application for a photo social network developed in Rails 4
== INSTALLATION: All you need is
* Ruby installation
* Rails installtion
* Mysql installation (local)
* configure database	
* bundle install
* rake db:migrate
== MAIN FEATURE:
* Login
* Logout
* Signup
* Upload Photos | Post
* Comment
* Comment with hashtags
* Search with hashtags
== UPCOMMING FEATURE:
* Real Time notification
* Follow
* Crop|Edit|Add Effect Photo
